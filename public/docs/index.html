Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

Create a new user account.

This endpoint allows users to create a new account. The user must provide their NPI, email, and password.

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"user@example.com\",
    \"npi\": 123456789,
    \"password\": \"#7u5v1=e\'IY$u[aU3\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "user@example.com",
    "npi": 123456789,
    "password": "#7u5v1=e'IY$u[aU3"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "success": true,
    "message": "Compte créer avec succès",
    "body": ""
}

Example response (404):


{
    "success": false,
    "message": "Npi incorrect.",
    "body": ""
}

Update a candidate.

This endpoint allows updating the details of a specific candidate. Changes cannot be made after the election's start date.

Example request:
curl --request POST \
    "http://localhost/api/candidat/1" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "npi=123456789"\
    --form "election_id=1"\
    --form "description=Une bonne description"\
    --form "name=John Doe"\
    --form "image=@C:\Users\hp ProBook 450 G8\AppData\Local\Temp\phpAE71.tmp" \
    --form "photo=@C:\Users\hp ProBook 450 G8\AppData\Local\Temp\phpAFD9.tmp" 
const url = new URL(
    "http://localhost/api/candidat/1"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('npi', '123456789');
body.append('election_id', '1');
body.append('description', 'Une bonne description');
body.append('name', 'John Doe');
body.append('image', document.querySelector('input[name="image"]').files[0]);
body.append('photo', document.querySelector('input[name="photo"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
